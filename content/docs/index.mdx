---
title: What is TapState?
description: Capture, transform, and serve operational data in one declarative workflow.
---

TapState is an open-source data platform for turning database changes into live operational state.

It brings three steps into one declarative workflow:

- **Capture** changes from operational data sources.
- **Transform** data as it moves.
- **Serve** fresh state to applications, APIs, and AI agents.
